# 卡师模拟器 · 知识图谱 v1.0

## 📦 核心类

### CardData
```
CardData (class_name, extends RefCounted)
├── name: String
├── fields: Array[String]          # 字段标签，如["魏","攻击"]
├── attack: int                     # 攻击力
├── health: int                     # 生命值
├── speed: int                      # 速度
├── permanent: bool                 # true=怪兽卡，false=魔法/技能
├── attackable: bool
├── can_attack: bool
├── card_effects: Array             # [effect_dict, ...] 效果列表
├── card_type: String               # "怪兽" / "魔法" / "陷阱"
├── energy: float                   # 能量消耗
├── created_at: String
├── get_preview_text() → String
├── get_effects_description() → String
├── get_full_effects_description() → String
├── load_from_dict(d: Dictionary)
├── to_dict() → Dictionary
└── static load_from_file(path) → CardData
```

### CardEffect
```
CardEffect (class_name, extends RefCounted)
├── trigger: String      # TR01-TR15 触发类型ID
├── action_type: String  # A01-A18 动作类型ID
├── action_value: int   # 动作数值
├── target_type: String # T01-T12 目标类型ID
├── constraints: Array[String]  # CO01-CO13 约束ID列表
├── get_short_description() → String    # "入场：伤害 3 → 敌方全体"
├── get_full_description() → String    # 完整描述
├── load_from_dict(d: Dictionary)
└── to_dict() → Dictionary

触发映射 TR:
  TR01=主动使用  TR02=入场  TR03=离场  TR04=攻击
  TR05=被攻击    TR06=受伤  TR07=破坏  TR08=回合开始
  TR09=回合结束  TR10=抽牌后  TR11=弃牌后  TR12=召唤成功
  TR13=魔法发动  TR14=战斗开始  TR15=生命值变化

动作映射 A:
  A01=伤害  A02=恢复  A03=破坏  A04=除外  A05=抽牌
  A06=弃牌  A07=召唤  A08=特殊召唤  A09=检索  A10=洗回
  A11=增减攻击  A12=增减生命上限  A13=增减速度
  A14=赋予字段  A15=移除字段  A16=赋予种族  A17=复制效果  A18=无效化

目标映射 T:
  T01=自身  T02=敌方单体  T03=友方单体  T04=敌方全体
  T05=友方全体  T06=随机敌方  T07=随机友方  T08=全场
  T09=手牌  T10=墓地  T11=卡组  T12=指定位置

约束映射 CO:
  CO01=一回合一次  CO02=一局一次  CO03=使用后破坏
  CO04=使用后除外  CO05=消耗生命  CO06=弃手牌  CO07=消耗能量
  CO08=仅己方回合  CO09=仅对方回合  CO10=需盖放
  CO11=需满足条件  CO12=非永久  CO13=不可攻击
```

### CardInstance
```
CardInstance (class_name, extends Resource)
├── card_data: CardData       # 所属卡牌数据
├── owner: int                 # 0=玩家, 1=AI
├── zone: String               # "hand" / "field" / "graveyard" / "excluded" / "deck"
├── current_attack: int        # 当前攻击力（含buff）
├── current_health: int        # 当前生命值
├── current_speed: int        # 当前速度
├── max_health: int            # 最大生命值
├── summon_sickness: bool      # 召唤失调（本回合不能攻击）
├── can_attack_this_turn: bool
├── used_constraints: Dictionary  # {"CO01": 1} 记录已用次数
├── modifiers: Dictionary        # 临时buff/debuff
├── unique_id: int              # 全局唯一实例ID
├── take_damage(amount) → bool  # 返回true表示死亡
├── heal(amount)
├── modify_attack(delta)
├── modify_health(delta)
├── is_alive() → bool
├── can_summon() → bool
└── refresh_turn()
```

### BattleManager

```
BattleManager (extends Node)
├── player: BattleSide
├── enemy: BattleSide
├── current_turn: int       # 0=玩家回合, 1=AI回合

├── is_running: bool

信号:
├── battle_end(winner: int)      # 0=玩家胜利, 1=AI胜利
├── turn_changed(current_turn)
├── log_message(msg: String)

方法:
├── start_battle(player_deck, enemy_deck)
├── end_turn()

├── attack(attacker, defender)     # 战斗
├── direct_attack(attacker)       # 直接攻击玩家
├── destroy_card(card)            # 破坏→墓地
├── exclude_card(card)           # 除外
├── get_all_cards() → Array[CardInstance]

├── process_ai_turn()
├── finish_battle(winner)
└── update_ui()
```

### EffectExecutor
```
EffectExecutor (class_name, extends Node)
├── battle: BattleManager  # 引用

方法:
├── static get_trigger_event(trigger_id) → String
│   # TR01→"active_use", TR02→"enter_field", TR04→"attack"...
├── execute_action(action_type, action_params, source, targets)
│   # A01=伤害, A02=治疗, A03=破坏, A04=除外,
│   # A05=抽牌, A06=弃牌, A07=召唤, A11=改攻击,
│   # A12=改生命, A18=无效化
├── select_targets(target_spec, source) → Array[CardInstance]
│   # T01=自身, T02=敌方单体, T04=敌方全体,
│   # T06=随机敌人, T07=攻击最低敌人...
├── check_condition(condition, source, target) → bool
│   # health_above, health_below, own_health_below...
├── trigger_effects(event_name, source, context)
│   # 遍历所有卡牌，检查触发条件，执行效果
└── check_constraints(constraints, card) → bool
    # CO01=一回合一次, CO02=一局一次...
```

### CraftManager (autoload)
```
CraftManager (extends Node, 全局单例)

阶段枚举 Phase:
  DESIGN=1      # 设计构思 → crafter.tscn
  MATERIAL=2    # 材料准备 → material.tscn
  BOTTOM_DRAW=3 # 制作基底 → draw_bottom.tscn
  NODE_PLACE=4  # 构建能量节点 → place_node.tscn
  LINE_DRAW=5   # 绘制能量回路 → draw_line.tscn
  ACTIVATE=6    # 封卡 → activate.tscn
  TEST=7        # 测试 → test.tscn

字段:
├── current_phase: Phase
├── current_card: CardData           # 阶段1产出
├── selected_materials: Dictionary    # 阶段2选择的材料
│   # {"paper":"PG02", "ink":"INK03", "core":"EC01", "aux":["AUX01"]}
└── phase_scores: Dictionary         # 各阶段评分

品质计算权重:
  设计 30% + 材料 20% + 基底 12.5% + 节点 12.5% + 回路 12.5% + 封卡 12.5%

品质等级:
  ≥90 → "完美"  ≥70 → "标准"  ≥50 → "瑕疵"  <50 → "废品"

方法:
├── calculate_final_quality() → float
├── quality_grade() → String
├── goto_phase(phase)
├── start_material()
├── set_card(card)
├── start_material_selection(card_data)
├── start_bottom_draw(materials)
├── start_node_place(score)
├── start_line_draw(score)
├── start_activate(score)
├── start_test(score)
├── finish_and_save() → bool  # 保存到 user://cards/
└── reset_crafting()
```

## 🔄 制作流程 (CraftManager)

```
阶段1 DESIGN ──→ [保存CardData到CraftManager] ──→ 阶段2 MATERIAL
                                                      │
  paper/ink/core/aux选择 ──→ start_bottom_draw() ──→ 阶段3 BOTTOM_DRAW
                                                          │
              绘制游戏得分 ──→ start_node_place() ──→ 阶段4 NODE_PLACE
                                                              │
                  放置节点游戏 ──→ start_line_draw() ──→ 阶段5 LINE_DRAW
                                                              │
                    绘制回路游戏 ──→ start_activate() ──→ 阶段6 ACTIVATE

                                                                      │
                                          finish_and_save() ──→ 保存到 user://cards/
```

## ⚔️ 战斗流程 (BattleManager)

```
start_battle()
  ├── 创建BattleSide(player/enemy)
  ├── 各自洗牌并抽3张
  └── 玩家先手 → start_turn()

玩家回合:
  1. start_turn() → 能量+3, 刷新场上怪兽
  2. 玩家操作: use_card() / attack() / direct_attack()
  3. 点击"结束回合" → end_turn()

end_turn():
  ├── trigger_effects("turn_end")
  ├── 切换current_turn
  ├── 抽1张牌
  └── AI回合: process_ai_turn()

process_ai_turn():
  ├── 消耗能量出牌（能量高的先出）
  └── 场上怪兽攻击（优先攻击攻击力最低的敌方）

攻击流程 attack(a, d):
  ├── 互相造成对方攻击力数值的伤害
  ├── trigger_effects("attack") / trigger_effects("be_attacked")
  ├── 死亡的卡 → destroy_card() → 墓地
  └── emit_signal("ui_update")

use_card():
  ├── 检查能量足够
  ├── 怪兽卡 → summon_to_field() + trigger("enter_field")
  └── 魔法卡 → execute_action() → send_to_grave()

每回合: 能量自动+3 (上限max_energy)
```

## 🎯 效果触发系统

```
效果字典结构:
{
  "trigger": "TR02",        # 入场触发
  "action": {"type": "A01", "value": 3},
  "target": {"type": "T04"},   # 敌方全体
  "constraints": ["CO01"]   # 一回合一次
}

触发链:
  事件发生 → trigger_effects(event_name, source)
    → 遍历场上所有卡牌的card_effects
    → 匹配trigger_id == event_name
    → check_constraints() 通过
    → select_targets() 获取目标
    → execute_action() 执行效果
    → emit_signal("log_message") 记录战报
```
